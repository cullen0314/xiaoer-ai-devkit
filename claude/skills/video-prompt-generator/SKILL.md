# 视频生成提示词生成器

## 核心定位

本技能用于**把用户的模糊视频想法整理成可直接使用的视频提示词**。

输出重点：
- 一条可直接复制使用的英文视频 prompt
- 一份结构化中文说明
- 可选的分镜头草案
- 可选的平台适配建议

边界说明：
- **负责**：视频提示词生成、分镜草案、风格化 prompt、提示词优化
- **不负责**：直接生成视频文件、复杂素材处理、完整视频解析流水线

---

## 使用方式

当用户提出视频创作需求时，优先按以下顺序工作：

### 第一步：先收用户原始需求
先让用户自然描述，不要一开始问太多问题。

可直接使用下面的话：

```text
请先用自己的话描述你想做的视频。

可以简单说这几件事里你已经想到的部分：
- 视频主题是什么
- 想要什么风格
- 大概几秒
- 打算发到哪里
- 是否需要分镜或文案
```

### 第二步：只补关键缺口
如果用户信息不完整，只补最关键的 2~4 项：
- 主体 / 主题
- 场景
- 风格
- 时长
- 平台
- 是否需要分镜

不要把技能做成问卷。

### 第三步：按需求选择输出类型
根据用户需求选择以下一种输出：

1. **标准 prompt**
   - 适合单条视频提示词需求
2. **分镜头草案**
   - 适合用户明确说要分镜、多镜头、短片脚本
3. **优化版 prompt**
   - 适合用户提供现有 prompt，希望优化
4. **参考风格版**
   - 适合用户说“类似某种风格/导演/作品”

---

## 推荐输入槽位

如果用户没有一次说全，可以按需补充以下信息：

| 槽位 | 说明 | 是否优先 |
|---|---|---|
| 主题 | 视频要讲什么 | 高 |
| 主体 | 人 / 动物 / 产品 / 场景 | 高 |
| 场景 | 室内 / 户外 / 城市 / 自然等 | 高 |
| 风格 | 写实 / 动画 / 广告 / 电影感等 | 高 |
| 时长 | 5 秒 / 15 秒 / 30 秒等 | 中 |
| 平台 | Sora / Runway / Kling / Pika / Veo / 通用 | 中 |
| 分镜 | 是否需要按镜头拆分 | 中 |
| 文案 | 是否需要旁白 / 字幕 / slogan | 低 |

## 平台适配原则

### 通用平台
优先输出稳定、清晰、可复制的英文 prompt。

### Runway / Kling / Pika / Sora / Veo
如用户指定平台，可在结果末尾补一句平台建议，例如：
- 更强调镜头运动
- 更强调主体动作清晰度
- 更强调场景和光影关键词
- 避免过长、过散的描述

如果平台不明确，默认输出 **通用版 prompt**。

---

## 分镜模式触发条件

以下情况适合输出分镜头草案：
- 用户明确说“分镜”“镜头脚本”“多镜头”
- 视频包含明显的前后阶段变化
- 用户要做广告片、剧情短片、产品展示片

如果用户没有明确要求，不必默认输出复杂分镜。

---

## 优化模式触发条件

如果用户已经给出 prompt，并且表达以下意思，则进入优化模式：
- “帮我优化这个提示词”
- “这个 prompt 太普通了”
- “改得更像广告片/电影感”
- “适配某个平台”

优化时优先做这些事：
- 提高主体清晰度
- 提高镜头表达清晰度
- 提高风格一致性
- 压缩冗余描述

## 输出原则

1. **先理解，再补问**：先接住用户原始想法
2. **只问关键项**：避免把 skill 用成长表单
3. **默认给可复制结果**：优先产出英文 prompt
4. **结构稳定**：中文摘要 + 结构化说明 + 英文 prompt
5. **按需增加复杂度**：只有用户要求时再展开分镜或细节

---

## 一句话执行准则

把用户的想法，整理成一份**简洁、专业、可直接使用的视频提示词结果**，而不是把流程做成复杂产品系统。
