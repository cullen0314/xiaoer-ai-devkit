# 视频生成提示词生成器

## Overview

本技能采用**"先输入后引导"**模式：先让用户自由描述想法，再解析描述识别缺失细节，进行针对性追问。适用于 Sora、Runway、Pika、Kling、Veo 等所有主流视频生成平台。

---

## 工作流程

### 第一步：识别触发词

当用户说以下内容时触发：
- "帮我写个视频生成提示词"
- "生成视频提示词"
- "AI视频提示词"
- "写一个xx风格的视频提示词"
- 其他类似表达

### 第二步：收集用户自由描述（第一步！）

**首先让用户自由描述他们的想法**：

```
请用你自己的话描述你想要的视频画面

可以简单也可以详细，想到什么说什么：
- 想要什么内容？
- 大概什么风格？
- 有什么特别想要的画面？
```

**重要**：不要在这里给出选项或引导，让用户自由表达！

### 第三步：解析描述 + 识别缺失维度

分析用户的描述，识别哪些维度缺失：

**必备维度检查清单**：

| 维度 | 检查问题 | 缺失时追问 |
|------|----------|------------|
| **主体身份** | 谁/什么？ | 具体是什么角色/动物/物体？ |
| **外观细节** | 长什么样？ | 颜色/体型/毛发/服饰？ |
| **动作行为** | 在做什么？ | 具体什么动作？怎么做的？ |
| **表情情绪** | 什么表情/情绪？ | 面部表情/姿态语言？ |
| **环境场景** | 在哪里？ | 室内/室外？什么场所？ |
| **风格定位** | 什么风格？ | 写实/动画/艺术风格？ |
| **镜头运动** | 什么镜头？ | 固定/运动？什么方式？ |
| **光影氛围** | 什么光？ | 什么时间/什么光影？ |

### 第四步：针对性追问主体细节

根据用户描述，**只追问缺失的维度**。

#### 4.1 主体外观详细追问

如果用户没有详细描述外观，追问：

```
请详细描述主体外观：

【基本特征】
- 颜色：什么颜色？
- 体型：胖/瘦/高/矮/标准？
- 毛发/发型：什么质感？

【服饰装扮】
- 穿着什么衣服？
- 有什么配饰/道具？
- 服装风格（现代/古装/奇幻/职业）？

【表情细节】
- 眼睛：大/小？什么颜色？眼神如何？
- 嘴巴：微笑/严肃/其他？
- 整体表情：开心/专注/惊讶/温柔？
```

#### 4.2 动作细节追问

如果用户没有详细描述动作，追问：

```
请描述具体动作细节：

【动作方式】
- 正在做什么具体动作？
- 动作速度：静止/缓慢/正常/快速？
- 动作幅度：大动作/小动作/微妙动作？

【互动关系】
- 与其他对象有互动吗？
- 与环境如何互动？
- 独自一人/有同伴？
```

#### 4.3 环境场景追问

如果用户没有详细描述环境，追问：

```
请描述环境场景：

【位置类型】
- 室内/室外/混合？
- 具体场所：房间/街道/自然/其他？

【环境元素】
- 背景有什么？
- 有什么标志性物品/建筑？
- 环境拥挤/空旷？

【天气/时间】
- 什么时间（清晨/白天/黄昏/夜晚）？
- 什么天气（晴朗/下雨/下雪/有雾）？
```

### 第五步：引导风格选择

如果用户没有明确风格偏好：

```
你想要什么视觉风格？

【写实风格】
A. 电影感写实（真实如纪录片）

【动画风格】
B1. 皮克斯/迪士尼（3D可爱，圆润）
B2. 宫崎骏/吉卜力（温暖手绘）
B3. 日式动漫（线条清晰，大眼风格）
B4. 水彩/油画（艺术质感）
B5. 黏土定格动画（有趣搞怪）

【科幻/奇幻】
C1. 赛博朋克（霓虹暗色调，高科技）
C2. 史诗奇幻（中土世界，魔法）
C3. 蒸汽朋克（黄铜机械，复古未来）

【复古风格】
D1. 80年代合成波（复古霓虹）
D2. VHS复古录像（怀旧噪点）
D3. 黑白电影（经典电影感）

【导演风格】
E1. 韦斯·安德森（对称构图，糖果色）
E2. 诺兰（史诗宏大，实拍特技）
E3. 库布里克（精准构图，冷色调）

【自定义】
F. 自己描述想要的风格
```

如需更多风格参考，读取 `references/styles.md`

### 第六步：引导镜头运动

如果用户没有描述镜头：

```
你想要什么样的镜头效果？

【镜头类型】
A. 建立镜头（广角全景→交代环境）
B. 推进镜头（靠近主体→聚焦细节）
C. 拉远镜头（远离主体→展示全貌）
D. 跟拍镜头（跟随主体移动）
E. 环绕镜头（围绕主体旋转）
F. 俯仰镜头（向上看/向下看）
G. 无人机航拍（俯瞰，飞越）
H. 特写镜头（聚焦面部/物体细节）
I. 组合镜头（多个镜头衔接）

【镜头节奏】
- 慢动作（Slow motion）
- 正常速度
- 快节奏/延时摄影
```

如需更多镜头术语，读取 `references/camera-terms.md`

### 第七步：引导光影氛围

如果用户没有描述光影：

```
你想要什么样的光影和氛围？

【时间段】
A. 黄金时刻（温暖日落光，温馨）
B. 蓝色时刻（日落后深蓝，宁静）
C. 正午阳光（强烈对比，活力）
D. 深夜（月光/城市灯光）

【光影效果】
E. 霓虹灯光（赛博朋克，彩色）
F. 体积光/丁达尔光（梦幻光束）
G. 轮廓光/逆光（戏剧性剪影）
H. 柔和漫射光（平静舒适）
I. 硬光对比（强烈明暗）

【氛围情绪】
J. 温馨治愈
K. 梦幻奇幻
L. 紧张悬疑
M. 史诗壮丽
N. 孤寂忧郁
O. 欢快活泼
P. 神秘诡异
```

如需更多光影词汇，读取 `references/lighting.md`

### 第八步：补充视觉细节（可选）

询问额外的视觉细节：

```
请补充视觉细节（可选）：

【色彩偏好】
- 暖色调（橙/黄/红）→ 温暖舒适
- 冷色调（蓝/青/紫）→ 清冷疏离
- 高饱和度 → 鲜艳活泼
- 低饱和度 → 柔和复古

【画面质感】
- 清晰锐利 → 现代感
- 柔焦朦胧 → 梦幻感
- 胶片颗粒感 → 电影感
- VHS复古 → 怀旧感

【特殊效果】
- 粒子/光点（魔法效果）
- 烟雾/云雾（氛围感）
- 火花/火焰（戏剧效果）
- 水/雨/雪（天气效果）
- 跳过（使用默认）
```

### 第九步：音频需求（可选）

```
需要描述音效/配乐吗？

A. 不需要（纯视觉）
B. 环境音（雨声/风声/城市噪音/脚步声）
C. 背景音乐（描述音乐风格和情绪）
D. 特定音效（爆炸/对话/动物叫声）
```

### 第十步：确认技术参数

询问时长、比例、平台：

```
【技术参数】
- 时长：5秒 / 10秒 / 15秒 / 其他
- 比例：16:9横屏 / 9:16竖屏 / 1:1正方形 / 2.35:1超宽屏
- 帧率：24fps电影感 / 30fps标准 / 60fps流畅

【目标平台】
- Sora
- Runway (Gen-2/Gen-3)
- Pika Labs
- Kling (可灵)
- Veo
- 通用/不确定
```

如需参数参考，读取 `references/aspect-ratios.md`

## 特殊处理

### 用户描述非常详细
- 识别已有的维度
- 只追问缺失的1-2项
- 避免重复询问

### 用户描述非常简单
- 逐步追问所有维度
- 每次只问1-2个问题
- 提供示例选项引导

### 用户有参考视频/图片
- 询问参考内容
- 提取风格/镜头/光影
- 融入新提示词

### 用户需要多版本
- 基于同一主题
- 在风格/镜头/光影上差异化
- 生成2-3个变体

## 分镜头脚本模式

### 触发条件

当用户说以下内容时，进入分镜头模式：
- "生成分镜头脚本"
- "视频分镜头"
- "多场景视频"
- "长视频/完整故事"
- "需要多个镜头"
- 或用户描述包含多个场景/情节变化

### 分镜头工作流程

#### S1：识别分镜头需求

分析用户描述，判断是否需要分镜头：

**需要分镜头的信号**：
- 描述包含多个场景/地点
- 描述有时间顺序（先...然后...）
- 描述有情节发展/故事线
- 提到"转场"、"切换到"
- 预计时长超过15秒

#### S2：确认镜头数量

询问用户需要多少个镜头：

```
你的视频需要分解成几个镜头？

A. 2-3个镜头（简单视频）
B. 4-5个镜头（中等复杂）
C. 6-8个镜头（复杂叙事）
D. 自定义数量
```

#### S2.5：收集主体/角色/风格详细信息

**在逐个镜头引导之前，先收集所有镜头共享的详细信息**：

##### S2.5.1：确认主体数量

```
视频中有几个主要角色/主体？

A. 单个主体（一个人/动物/物体）
B. 两个主体（对话/互动）
C. 多个主体（3个以上，群体场景）
D. 不确定/随镜头变化
```

##### S2.5.2：逐个收集主体详细信息

**如果选择A（单个主体）**：

```
请描述主体信息：

【主体类型】
- 是什么角色/动物/物体？具体是什么？

【外观细节】
- 颜色/体型/毛发/发型是什么？
- 穿着什么衣服？有什么配饰道具？

【表情与性格】
- 什么表情？什么性格特点？
```

**如果选择B或C（多个主体）**：

对每个主体，依次收集：

```
【主体 1】
- 名称/角色：[例如：小明、店主、路人A]
- 类型：[人/动物/物体]
- 外观：[颜色、体型、服饰、配饰]
- 性格/表情：[性格特点、常见表情]

【主体 2】
- 名称/角色：[例如：小红、顾客、路人B]
- 类型：[人/动物/物体]
- 外观：[颜色、体型、服饰、配饰]
- 性格/表情：[性格特点、常见表情]

【主体 3】（如有）
...
```

**主体关系确认**：

```
这些主体之间是什么关系？

A. 对话关系（有交流互动）
B. 对立关系（冲突/竞争）
C. 陪伴关系（朋友/伙伴）
D. 陌生关系（路人/偶遇）
E. 其他关系：[描述]
```

##### S2.5.3：收集对话内容（如适用）

**如果主体有对话关系**：

```
视频中有对话吗？

A. 有对话（需要详细描述）
B. 有简单交流（声音/表情）
C. 无对话（纯动作）
```

**如果选择A（有对话）**：

```
请描述对话内容：

【对话方式】
A. 同期声（真实配音）
B. 字幕形式（画面显示字幕）
C. 气泡对话框（漫画风格）

【主要对话】
请按镜头或场景列出对话内容：

镜头1：
- [主体A]：[对话内容]
- [主体B]：[对话内容]

镜头2：
- [主体A]：[对话内容]
- [主体B]：[对话内容]

...

【对话风格】
- 语气：[轻松/严肃/激动/温柔]
- 语言风格：[日常/正式/幽默/文艺]
```

##### S2.5.4：收集动作描述

```
请描述视频中的主要动作：

【动作类型】
A. 日常动作（行走、吃饭、工作）
B. 表演动作（舞蹈、武术、表演）
C. 交互动作（对话、打斗、拥抱）
D. 特殊动作（魔法、超能力、特效）

【主要动作序列】
请描述视频中的关键动作：

动作1：[主体A] [做什么] [如何做] [快/慢]
动作2：[主体B] [做什么] [如何做] [快/慢]
动作3：[互动描述] [如：A递给B某物]

【动作细节】
- 动作速度：[静止/缓慢/正常/快速/变速]
- 动作幅度：[大动作/小动作/微妙动作]
- 动作风格：[流畅/有力/笨拙/优雅]
```

##### S2.5.5：环境/场景信息

```

【轻快活泼】推荐：

B. 跳跃剪辑（Jump Cut）- 轻盈跳跃感

【紧凑动感】推荐：
A. 快速剪辑（Rapid Cut）- 节奏紧凑
B. 运动模糊过渡（Motion Blur Wipe）- 速度感
C. 甩镜头（Whip Pan）- 快速横摇过渡

【起伏变化】推荐：
A. 组合使用多种转场
B. 根据情节节奏调整
```

### 专业转场类型详解

| 转场类型 | 英文 | 适用场景 | 视觉效果 |
|---------|------|----------|----------|
| **硬切** | Hard Cut | 动作场面、对话场景 | 直接切换，干脆有力 |
| **溶解** | Dissolve | 情感场景、时间流逝 | 画面渐变重叠，柔和 |
| **交叉溶解** | Cross Dissolve | 回忆、梦境过渡 | 两画面短暂混合 |
| **淡入淡出** | Fade to/from Black | 章节/段落分隔 | 画面渐暗至黑/亮起 |
| **擦除** | Wipe | 空间转换、并列对比 | 侧向/径向擦除切换 |
| **伸缩转场** | Push/Pull Slide | 场景切换推入推出 | 新镜头推入旧镜头 |
| **匹配剪辑** | Match Cut | 形状/动作相似转换 | 视觉元素连续 |
| **跳跃剪辑** | Jump Cut | 时间压缩、节奏跳跃 | 同场景突然变化 |
| **甩镜头** | Whip Pan | 快速场景转换 | 横向快速模糊移动 |
| **运动模糊** | Motion Blur Wipe | 动作场面过渡 | 运动方向模糊切换 |
| **焦外切换** | Rack Focus | 同场景焦点转移 | 前后景焦点变化 |
| **缩放转场** | Zoom Transition | 强调/戏剧性 | 快速推入/拉出切换 |
| **闪白/闪黑** | Flash to White/Black | 高潮/梦境/回忆 | 瞬间白场/黑场 |
| **遮挡转场** | Mask Transition | 有趣的创意转场 | 物体经过遮挡切换 |
| **粒子过渡** | Particle Dissolve | 魔法/奇幻场景 | 粒子分解重组 |

### 镜头运动与转场配合

```
【镜头运动 + 转场 = 自然流畅】

【推进 → 溶解】
前镜头缓慢推进 + 溶解过渡 = 自然衔接
适用：远景→中景，情绪渐进

【环绕 → 硬切】
前镜头环绕结束 + 硬切 = 动态切换
适用：动作场面，节奏转换

【拉远 → 淡出】
前镜头拉远 + 淡出 = 优雅结束
适用：场景结束，段落收尾

【跟拍 → 运动模糊】
前镜头跟拍 + 运动模糊过渡 = 速度延续
适用：追逐场景，连贯运动

【固定 → 交叉溶解】
前镜头固定 + 交叉溶解 = 时间流动
适用：时间流逝，场景变化
```

### 分镜头模式原则

1. **风格统一**：所有镜头保持一致的风格和色调
2. **逻辑衔接**：镜头之间有逻辑关系
3. **时长合理**：根据内容重要程度分配时长
4. **转场自然**：选择合适的转场方式，让镜头衔接流畅和谐

### 转场衔接核心原则

**1. 转场类型一致性**
- 主视频使用 1-2 种主要转场方式
- 重要情节转折点使用特殊转场
- 避免每个镜头都用不同转场

**2. 转场与节奏匹配**
- 舒缓场景：溶解、淡入淡出（0.5-1秒）
- 轻快场景：硬切、跳跃剪辑（0.3-0.5秒）
- 紧凑场景：快速剪辑、运动模糊（0.2-0.3秒）

**3. 转场与镜头运动配合**
- 推进镜头 → 溶解：自然渐进
- 拉远镜头 → 淡出：优雅收尾
- 跟拍镜头 → 运动模糊：延续速度感
- 固定镜头 → 交叉溶解：时间流动

**4. 转场与情绪匹配**
- 温馨治愈 → 溶解、交叉溶解
- 紧张悬疑 → 硬切、快速剪辑
- 梦幻回忆 → 柔焦过渡、闪白
- 史诗壮丽 → 慢推拉 + 溶解

**5. 场景关系决定转场**
- 同场景景别变化 → 溶解/硬切
- 不同场景切换 → 溶解/擦除/伸缩
- 动作连续 → 硬切/运动匹配
- 时间跳跃 → 溶解/淡入淡出
