# 常用场景镜头预设

本文档包含常见视频场景的标准镜头组合和转场方案，可直接套用。

---

## 日出/日落场景

### 标准日出流程
```
镜头1（2秒）：天空泛起鱼肚白，地平线微光
镜头2（3秒）：太阳半露，金光初现
镜头3（4秒）：太阳完全升起，金光洒满大地
镜头4（3秒）：特写受光照亮的细节（露珠、树叶）

转场：溶解过渡，流畅自然
色调：蓝紫→橙金渐变
音乐：渐强的环境音
```

### 黄昏时分流程
```
镜头1（3秒）：太阳低垂天空，拉长影子
镜头2（4秒）：暖橙色光线覆盖场景
镜头3（3秒）：蓝色时刻降临，灯光亮起
镜头4（2秒）：夜景完全呈现

转场：缓慢溶解
色调：金黄→深蓝
氛围：从温暖到宁静
```

## 使用建议

1. **直接套用**：选择合适的场景模板，填入你的主体和细节
2. **灵活调整**：根据实际需要增减镜头数量和时长
3. **组合使用**：多个场景模板可以组合成更长的视频
4. **风格适配**：在模板基础上添加风格描述和光影设定

### 快速生成示例

以"宠物猫咪日常"为例：

```
镜头1（2秒）：猫咪在猫窝里睡觉，蓬松柔软
镜头2（3秒）：醒来伸懒腰，打哈欠
镜头3（4秒）：走向食盆，开始吃饭
镜头4（3秒）：饭后舔毛，悠闲自在
镜头5（2秒）：又趴下睡觉，可爱定格

风格：温暖治愈，宠物视频
色调：暖色调，明亮舒适
比例：9:16竖屏
总时长：14秒
```
