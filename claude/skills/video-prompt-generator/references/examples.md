## 示例一：赛博朋克城市夜景

### 中文描述

```json

  "style": "赛博朋克, 霓虹美学, 银翼杀手风格",
  "camera": "建立镜头(广景)→缓慢推进→平摇展示城市",
  "lighting": "粉红色和青色霓虹灯, 湿润街面反射光, 体积雾",
  "atmosphere": "雨夜, 神秘, 反乌托邦",
  "duration": "10秒",
  "ratio": "16:9"
}
```

### 英文提示词（可复制）
```
A futuristic cyberpunk cityscape at night, neon-lit streets with pink
and cyan signs reflecting on rain-slicked pavement, Blade Runner
aesthetic, establishing wide shot slowly pushing in toward central
skyscraper, camera panning to reveal the extent of the city, volumetric
fog, bokeh car lights, cinematic color grading, 16:9 widescreen,
10 seconds, 24fps film look, high contrast, mysterious atmosphere
```

## 提示词结构总结

一个完整的视频生成提示词通常包含以下部分（按重要性排序）：

1. **主体描述** - 核心内容，放在最前面
2. **风格/美学** - 决定整体视觉风格
3. **镜头运动** - 描述摄像机如何运动
4. **光影效果** - 营造氛围的关键
5. **情绪/氛围** - 描述观看感受
6. **技术参数** - 时长、比例、帧率（放在最后）

### 英文提示词写作技巧

| 技巧 | 说明 |
|------|------|
| **具体描述** | 用具体词汇而非抽象概念 |
| **感官词汇** | 描述视觉、听觉、触觉感受 |
| **动作动词** | 使用动态词汇描述运动 |
| **修饰词叠加** | 2-3个形容词叠加增强效果 |
| **参考作品** | 提及知名作品或导演风格 |

### 常用修饰词

| 效果 | 英文词汇 |
|------|----------|
| 加强语气 | highly, extremely, deeply, richly |
| 强调品质 | professional, cinematic, epic, masterpiece |
| 描述质感 | detailed, intricate, nuanced, subtle |
| 描述感受 | breathtaking, awe-inspiring, mesmerizing, haunting |
