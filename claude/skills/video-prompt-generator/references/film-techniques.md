# 电影技术参考库

本文档包含视频制作中的电影技术和剪辑术语。

---

## 剪辑技巧

| 技巧 | 英文 | 说明 |
|------|------|------|
| **匹配剪辑** | Match cut | 相似形状/动作的场景转换 |
| **跳跃剪辑** | Jump cut | 时间/空间跳跃 |
| **交叉剪辑** | Cross-cutting, Parallel editing | 同时展示多个场景 |
| **溶解过渡** | Dissolve, Fade transition | 渐变过渡 |
| **擦除过渡** | Wipe transition | 擦除式过渡 |
| **闪回** | Flashback | 回到过去的时间点 |
| **闪前** | Flashforward | 跳到未来的时间点 |
| **时间膨胀** | Time dilation | 时间感知变化 |

## 使用建议

在提示词中引用这些技巧时，描述视觉效果而非技术术语：

```
Montage sequence showing a city transforming from day to night,
time-lapse clouds, lights turning on, smooth transitions between shots
```

```
Action sequence with rapid cutting, dynamic camera movements,
motion blur on fast movements, explosive visual effects
```

```
Dream sequence with soft focus, ethereal glow, slow motion,
dissolve transitions between surreal imagery
```
