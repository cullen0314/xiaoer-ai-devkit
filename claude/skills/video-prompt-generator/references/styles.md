## 使用建议

在提示词中组合使用风格描述，例如：

```
Anime style, Studio Ghibli inspired, warm watercolor texture,
hand-drawn aesthetic with soft edges and natural lighting
```

```
Cyberpunk cityscape, neon lights, rain-slicked streets,
high contrast, Blade Runner aesthetic, synthwave color palette
```
