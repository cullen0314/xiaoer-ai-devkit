# 金字塔原理 - 提示模板

## 快速模式提示模板

### 分析阶段

```
你是一位金字塔原理专家。请分析以下内容：

【内容】
{content}

【分析要求】
1. 检查是否符合金字塔原理四大原则：
   - 结论先行：开头是否有明确的核心观点
   - 以上统下：上层思想是否总结下层的
   - 归类分组：同类内容是否归为一组
   - 逻辑递进：是否有逻辑顺序

2. 诊断问题：
   - 结论是否明确
   - 论据是否支撑结论
   - 分类是否符合MECE原则
   - 结构是否清晰

【输出格式】
按照 SKILL.md 中定义的分析报告格式输出。
```

### 优化阶段

```
基于上述分析，请将内容重组为金字塔结构：

【优化要求】
1. 结论先行：将核心观点放在最前面
2. 以上统下：确保上层概括下层
3. MECE原则：确保分类相互独立、完全穷尽
4. 逻辑递进：按重要性/时间/结构顺序排列

【输出格式】
先展示优化后的内容，然后简要说明改动之处。
```

## SCQA 模型模板

```
【SCQA 序言生成模板】

S - Situation 情境：
描述当前稳定状态，大家都认可的背景事实。
"目前我们公司的产品年增长率为20%..."

C - Complication 冲突：
描述发生了什么变化或问题，打破稳定状态。
"但最近三个季度，增长率下降到5%..."

Q - Question 疑问：
基于情境和冲突，自然产生的问题。
"如何恢复增长？"

A - Answer 答案：
你的核心结论。
"我们需要开拓新市场，具体来说..."
```

## MECE 检查模板

```
【MECE 原则检查清单】

相互独立（Mutually Exclusive）：
□ 各分类之间没有重叠
□ 一个事物只属于一个分类
□ 没有概念包含关系的并列项

完全穷尽（Collectively Exhaustive）：
□ 覆盖了所有可能性
□ 没有"其他"这种模糊分类
□ 考虑了边界情况

【常见MECE错误】
❌ 时间重叠：儿童(0-12)、青少年(10-18) → 重叠在10-12
✅ 正确划分：儿童(0-12)、青少年(13-18)、成年人(19+)

❌ 概念包含：线上渠道、电商渠道 → 电商是线上的子集
✅ 正确划分：电商渠道、社交媒体渠道、线下渠道
```

## 逻辑顺序模板

```
【三种逻辑顺序】

1. 时间顺序
   - 第一步、第二步、第三步...
   - 过去、现在、未来
   - 阶段一、阶段二、阶段三

2. 结构顺序
   - 北京、上海、广州...（地理位置）
   - A部门、B部门、C部门...（组织结构）
   - 自上而下、由外而内...（空间结构）

3. 重要性顺序
   - 最重要、次重要、次要...
   - 按影响程度排序
   - 按紧急程度排序

【选择建议】
- 讲过程 → 时间顺序
- 讲结构 → 结构顺序
- 讲理由 → 重要性顺序
```
