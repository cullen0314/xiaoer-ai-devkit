# OpenClaw 高推荐技能清单

## 说明

本文档列出了 **高使用度** 和 **高推荐度** 的 OpenClaw 技能，按功能分类整理。推荐基于以下标准：
- 社区活跃度和使用广泛性
- 功能实用性和通用性
- 安全性和可靠性
- 主流服务集成

---

## 🔐 安全必备（强烈推荐安装）

| 技能名称 | 作用 | 推荐理由 |
|---------|------|----------|
| **1password** | 1Password CLI 集成，安全存储和管理密码 | 行业领先的密码管理器 |
| **bitwarden** | Bitwarden/Vaultwarden 密码管理 | 开源密码管理方案 |
| **credential-manager** | OpenClaw 的强制安全基础 | 代理凭证管理的基础组件 |
| **ggshield-scanner** | 检测 500+ 种硬编码密钥/机密 | 防止密钥泄露到代码仓库 |
| **skillguard-audit** | 安装前审计技能安全性 | 安全第一，预防恶意技能 |
| **skill-cleaner** | 通过 VirusTotal 验证可疑技能 | 自动化安全检查 |
| **safe-encryption-skill** | 现代化 GPG 替代品，支持后量子加密 | 加密通信必备 |
| **domain-trust-check** | 检查钓鱼、恶意软件、品牌滥用 | 访问 URL 前的安全检查 |
| **pincer** | 安装代理技能的安全优先包装器 | 技能安装安全网 |

## 📋 快速开始推荐

### 新手安装顺序（推荐）

1. **安全基础**（必备）
   - credential-manager
   - 1password 或 bitwarden

2. **开发效率**（开发者推荐）
   - auto-test-generator
   - astrai-code-review
   - agent-team-orchestration

3. **AI 优化**（降低成本）
   - astrai-inference-router
   - ai-prompt-gen

4. **日常协作**
   - clawemail 或 microsoft365
   - briefing（每日简报）

5. **浏览器自动化**（按需）
   - Agent Browser
   - playwright
